```markdown
## From Bits To Coordinates

The QTT is built over binary site indices. Each site value is either `1` or
`2`, representing a bit value `0` or `1`. The following small functions keep
that mapping visible.
```

Add a Julia code cell:

```julia
function index_to_bit_values(i, R)
    index0 = i - 1
    return [((index0 >> (k - 1)) & 1) + 1 for k in 1:R]
end

function bit_values_to_grid_index(values)
    index0 = 0
    for (k, value) in enumerate(values)
        bit = Int(value) - 1
        index0 += bit * 2^(k - 1)
    end
    return index0 + 1
end

function bit_values_to_x(values; xmin, xmax, R)
    grid_index = bit_values_to_grid_index(values)
    return xmin + (grid_index - 1) * (xmax - xmin) / (2^R - 1)
end

(index_1_bits=index_to_bit_values(1, R), index_last_bits=index_to_bit_values(npoints, R))
```

```julia
const TN = Tensor4all.TensorNetworks

function qtt_for_function(f; R, xmin, xmax, tolerance, maxbonddim)
    local_dimensions = fill(2, R)
    sample_on_bits = bit_values -> f(bit_values_to_x(bit_values; xmin, xmax, R))

    tci = Tensor4all.TensorCI.crossinterpolate2(
        Float64,
        sample_on_bits,
        local_dimensions;
        tolerance=tolerance,
        maxbonddim=maxbonddim,
    )

    simple_tt = Tensor4all.SimpleTT.TensorTrain(tci)
    sites = [Index(2; tags=["bit", "bit=$k"]) for k in 1:R]
    qtt = TN.TensorTrain(simple_tt, sites)

    return (; tci, simple_tt, qtt, sites)
end

tolerance = 1e-12
maxbonddim = 20

cosh_qtt = qtt_for_function(
    target_function;
    R=R,
    xmin=xmin,
    xmax=xmax,
    tolerance=tolerance,
    maxbonddim=maxbonddim,
)

(sites=length(cosh_qtt.sites), tolerance=tolerance, maxbonddim=maxbonddim)
```

- [ ] **Step 12: Add the hand-calculation explanation**

Add a Markdown cell:

```markdown
## Why `cosh(x)` Is A Good First Example

The function `cosh(x)` can be written as

    cosh(x) = (exp(x) + exp(-x)) / 2

On a quantics grid, the binary representation of the grid index decomposes the
coordinate into a sum of bit contributions. Exponentials of sums factor into
products of exponentials. This is why exponential functions, and therefore
`cosh(x)` as a sum of two exponentials, admit a very compact QTT
representation.

The important observation for now is not the full derivation. It is that some
smooth structured functions can have very small QTT bond dimensions.
```
